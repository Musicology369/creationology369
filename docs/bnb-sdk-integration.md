# BNB Agent SDK Integration Guide

All agents in CREATIONology369 use the official BNB Agent SDK (mainnet live).

**Key Features Used:**
- ERC-8004 Identity
- ERC-8183 Commerce & Escrow
- MPP + x402 autonomous payments
- BNB Greenfield memory & storage

Every agent inherits from `GivingEconomyAgent` to enforce circular principles.
